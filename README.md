## 📋 Sobre
Esse é o roadmap de Engenharia de Dados, Análise de Dados e Ciência de Dados, baseado em fundamentos, principais tecnologias de mercado e projetos práticos do mundo real. Este repositório contém todo o conteúdo prático, projetos, exercícios e materiais de apoio utilizados durante meu aprendizado.

## 🛠️ Data Engineering Roadmap
Este repositório é o meu caderno de estudos focado no ciclo de vida do dado. Aqui compartilho conteúdos, exercícios, projetos e anotações organizados por temas, integrando a infraestrutura da Engenharia com a inteligência da Análise e Ciência.

## 🚀 Objetivo
- Engenharia de Dados: aprender conceitos essenciais de ETL/ELT, modelagem de dados, Data Warehouses, Data Lakehouse e boas práticas na construção de pipelines.
- Ciência de Dados: dominar a Análise Exploratória de Dados (EDA), visualização de dados com Seaborn/Plotly, além de algoritmos de Machine Learning para modelos preditivos e classificação.
- Análise de Dados: explorar o uso de Python, SQL, Pandas, Excel e Power BI para gerar insights e realizar Estatística Aplicada ao negócio.
- Prática: construir projetos práticos ponta a ponta, desde a ingestão dos dados até a entrega de modelos e dashboards.
